## ☕ About The Project

<img align="right" width="300" src="https://media.giphy.com/media/v1.Y2lkPTc5MGI3NjExcDd6NnA3ZmYxbW9yajE4a2FvaTlzbGw1cWxwdnYyZ2tkcGZqZDd6ZSZlcD12MV9pbnRlcm5hbF9naWZfYnlfaWQmY3Q9Zw/xT0xeMA62E1XIlqNb2/giphy.gif" alt="Coffee Animation"/>


> *"Behind every great program is a substantial amount of freshly ground coffee."*

**The Brew Haven** is a warm, minimalist e-commerce & reservation web application designed for modern cafes. Built with a cozy earthy aesthetic, real-time table booking, interactive custom drink builders, and seamless ordering ---🌟


## ✨ Signature Features

- 🫘 **Interactive Drink Customizer** — Select roast, milk ratio, sweetness, and toppings with live visual feedback.
- 🥐 **Artisanal Bakery Pairing** — AI-driven pastry pairings based on your coffee choice.
- 💳 **Speedy Pour Checkout** — Fast, encrypted payments via Stripe & Apple Pay.
- 🪑 **Cozy Seat Reservations** — Pick your spot by the window or near a charging outlet.
- 🌙 **Warm Latte Dark Mode** — Designed with eye-comfort warm sepia & mocha tones.
